# 小乐AI文件存储目录

这个目录用于存储通过小乐AI文件工具创建和管理的文件。

## 📁 目录说明

- **用途**: 存储用户通过小乐创建的文件（笔记、待办、配置等）
- **持久化**: 文件会持久保存，不会因系统重启而丢失
- **安全**: 仅允许特定文件类型，受文件工具安全限制保护

## 🔒 安全限制

### 支持的文件类型
- 文本: `.txt`, `.md`, `.log`
- 配置: `.json`, `.yaml`, `.yml`, `.xml`
- 代码: `.py`, `.js`, `.html`, `.css`
- 数据: `.csv`

### 文件大小限制
- 最大 10MB

## 💡 使用方法

直接对小乐说：

```
"帮我创建一个shopping.txt，内容是：牛奶、鸡蛋、面包"
"读取notes.md的内容"
"列出所有文件"
```

所有文件都会自动保存在这个目录下。

## 📝 示例

创建待办清单：
```
你：创建todo.txt，内容是：
1. 学习Python
2. 完成项目
3. 写文档

小乐：✅ 已创建文件 todo.txt
     位置: files/todo.txt
```
