## 测试技巧

1. **等待时间**：每次对话后等待2-3秒，让系统分析
2. **观察标记**：追问会有`💭`前缀
3. **记录结果**：每次测试后记录是否触发
4. **避免干扰**：测试时不要说"别问了"等不耐烦词汇
