# 键盘快捷键指南 (v0.6.0)

小乐AI管家现在支持丰富的键盘快捷键，提升操作效率！

## 🎹 快捷键列表

### 聊天操作
| 快捷键 | Mac | 功能 | 说明 |
|--------|-----|------|------|
| `Ctrl+Enter` | `⌘+Enter` | 发送消息 | 在聊天标签页时生效 |
| `Enter` | `Enter` | 发送消息 | 默认行为（Shift+Enter换行） |
| `Esc` | `Esc` | 清空输入框 | 输入框聚焦时生效 |

### 导航操作
| 快捷键 | Mac | 功能 | 说明 |
|--------|-----|------|------|
| `Ctrl+K` | `⌘+K` | 新建对话 | 快速开始新对话 |
| `Ctrl+/` | `⌘+/` | 切换到聊天 | 快速回到聊天界面 |
| `Ctrl+1` | `⌘+1` | 聊天标签 | 快速切换到聊天 |
| `Ctrl+2` | `⌘+2` | 会话列表 | 查看历史会话 |
| `Ctrl+3` | `⌘+3` | 记忆管理 | 查看记忆库 |
| `Ctrl+4` | `⌘+4` | 行为分析 | 查看分析报告 |
| `Ctrl+5` | `⌘+5` | 提醒管理 | 管理提醒 |
| `Ctrl+6` | `⌘+6` | 工具管理 | 查看工具 |

## 💡 使用技巧

### 1. 快速发送消息
输入完消息后，无需点击发送按钮，直接按 `Ctrl+Enter`（Mac: `⌘+Enter`）即可发送。

### 2. 多行输入
需要换行时，使用 `Shift+Enter`，完成后按 `Ctrl+Enter` 发送。

### 3. 快速清空
输入错误想重新输入？按 `Esc` 键快速清空输入框。

### 4. 标签导航
使用 `Ctrl+数字` 快速在不同功能间切换，提升工作效率。

### 5. 新对话
随时按 `Ctrl+K` 开始新的对话，无需鼠标点击。

## 🎨 视觉提示

- **提示栏**: 输入框聚焦时，底部会显示快捷键提示栏
- **自动隐藏**: 失焦2秒后自动隐藏，不影响界面
- **首次提示**: 页面加载后会显示3秒，帮助新用户了解功能

## 📱 移动端支持

移动端不支持键盘快捷键，但提供：
- 触摸手势支持
- 快速操作按钮
- 优化的触摸界面

## ⚙️ 技术实现

- **全局监听**: 使用 `document.addEventListener('keydown')`
- **系统检测**: 自动识别 Mac/Windows/Linux
- **事件优先级**: 阻止默认行为，避免冲突
- **条件触发**: 根据当前标签页智能触发

## 🐛 已知问题

1. 部分浏览器插件可能拦截快捷键
2. 输入法激活时，部分快捷键可能无效
3. 全屏模式下提示栏可能被遮挡
